## 1. System Overview
This application is an exclusive gamified learning system designed only for officially enrolled PUP TechVoc students to enhance culinary skills through interactive modules and simulations.

## 2. Target Users
- PUP TechVoc Students  
- Instructors (Monitoring only)  
- Admin (Verification & Content Management)

## 3. Access Control
Students must register using their official PUP Student Number. Verification is performed using a preloaded TechVoc masterlist database.

## 4. Core Modules
- Basic Cooking Skills  
- Food Safety & Hygiene  
- Kitchen Equipment  
- Recipe Challenges  
- Nutrition Basics  

## 5. Game Mechanics
- Points & Badges  
- Unlock Levels  
- Retry & Review Mode  
- Skill Assessments  

## 6. Removed Features
- LMS Integration  
- Cost Management Module  

## 7. Admin Panel
- Student Registration Approval  
- Content Management  
- System Analytics  
- Gamification Settings  

## 8. Offline Features
- Downloadable Content  
- Local Progress Tracking  
- Auto Sync when Online  

## 9. Security
Only verified PUP TechVoc students can access the application.
